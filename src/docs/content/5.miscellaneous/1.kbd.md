# Kbd

## Classes Information

| ClassName  | Description      |
| ---------- | ---------------- |
| **kbd**    | Base Class       |
| **kbd-xs** | Extra small size |
| **kbd-sm** | Small size       |
| **kbd-md** | Medium size      |
| **kbd-lg** | Large size       |
| **kbd-xl** | Extra large size |

## Basic

::kbdcom
---
basic: true
---
::

### Example Code

```html [html]
<kbd class="kbd">R</kbd>

```
