# Divider

## Classes Information

| ClassName              | Description                 |
|------------------------|-----------------------------|
| **divider**            | Base class                  |
| **divider-vertical**   | Set vertical orientation    |
| **divider-horizontal** | Base horizontal orientation |

## Basic

::divider

